Naming Conventions

Folders should be named following the YYYY_MM_DD format (ex - 2026_01_15). 

If more than one experiment was done per day, add letters following the date (ex - 2026_01_15a, 2026_01_15b, 2026_01_15c). 

All images should be in .jpeg format. The images themselves need to be named in a specific manner as well. 

Scale images with the 10x objective should be named a_scale.jpeg. 

Scale images with the 4x objective should be named b_scale.jpeg. 

Copepod image names should start with either a_ or b_, indicating which scale should be used when calibrating ImageJ.

After the scale indicator, images should be named after the tube number (ex - a_tube_01.jpeg, b_tube_10.jpeg).
- Note: Make sure to pad the number (use tube_01 instead of just tube_1).
